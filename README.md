# 图片隐私水印工具 · PRD

## 一、产品概述

一款运行在浏览器端的图片批量水印工具。用户上传图片后，可自定义水印文字，系统自动将水印叠加到原图上，保持原始尺寸与质量，最终批量下载处理后的图片。

---

## 二、目标用户

| 用户类型 | 使用场景 |
|---|---|
| 个人用户 | 对证件照/合同扫描件加水印防盗用 |
| 企业员工 | 内部文件流转时标注「内部资料/禁止外传」 |
| 摄影师 | 作品样图批量添加版权水印 |

---

## 三、核心功能需求

### F1 · 图片批量上传

- 支持 **拖拽上传** 和 **点击选择** 两种方式
- 支持格式：`JPG` / `PNG` / `WEBP`
- 单次最多上传 **100 张**
- 上传后展示缩略图列表，可单独移除某张

### F2 · 水印配置（核心）

| 配置项 | 说明 | 默认值 |
|---|---|---|
| 水印文字 | 用户自由输入，支持多行 | `「内部资料·禁止外传」` |
| 字体大小 | 滑块调节，范围 12–72px | 24px |
| 字体颜色 | 预设 5 种常用颜色 + 自定义拾色器 | 灰色 `#CCCCCC` |
| 透明度 | 滑块调节，范围 10%–100% | 30% |
| 排列方式 | 平铺（推荐）/ 单个居中 | 平铺 |
| 旋转角度 | 滑块调节，范围 -90°–0° | -45° |
| 间距密度 | 水印之间的间距（稀疏 / 适中 / 密集）| 适中 |

### F3 · 实时预览

- 配置面板任一参数变更时，右侧预览区**实时渲染**水印效果
- 预览基于列表中第一张图片（或用户指定某张）

### F4 · 图片输出

- 输出格式与输入保持一致（JPG 输出 JPG，PNG 输出 PNG）
- **不压缩、不缩放**，保持原始像素尺寸
- JPG 质量固定 100%
- 支持 **逐张下载** 和 **一键打包下载（ZIP）** 两种方式
- 输出文件命名规则：`原文件名_watermarked.扩展名`

### F5 · 处理进度

- 批量处理时展示进度条（当前第 X 张 / 共 Y 张）
- 处理完成后弹出提示，提供下载入口

---

## 四、非功能需求

| 维度 | 要求 |
|---|---|
| 性能 | 单张 ≤10MB 的图片处理耗时 < 2 秒 |
| 隐私安全 | 图片**全程在本地（浏览器端）处理**，不上传至任何服务器 |
| 兼容性 | Chrome / Safari / Firefox 最新正式版 |
| 响应式 | 适配桌面端（1280px+），移动端不作要求 |

## 六、核心用户流程

```
上传图片 → 配置水印参数 → 实时预览确认 → 点击「开始处理」→ 等待进度 → 下载
```
